# node-websockets
Exemplo de Chat em Node.js usando WebSockets

## Instalar dependências
```
    npm install
```

Para iniciar o servidor, use o comando:

```
   npm start
```
## Referencia
[Documentação do Socket.IO](https://socket.io)
